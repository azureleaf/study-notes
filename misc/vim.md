## MISC

- `.vimrc`
  - This is the config file for vim
  - `set number`
  - `set ts=4 sw=4`
    - Set the apparent tabstop size & shift width as 4 white spaces


## Edit

### copy & paste
- yy
  - copy the line
- p
  - paste
- dd
  - remove the line

### undo
- u
  - undo
- ctrl + r
  - redo
