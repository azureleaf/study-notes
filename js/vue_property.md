# 重要なプロパティ

## data:
    - 基本的な変数はここに書く
    - Vue インスタンスの場合は、変数を直接オブジェクトリテラルで書く
    - Vue Component の場合は、「オブジェクトリテラルを return する関数」

## computed:
- getter として働くのが基本だが、自分で定義すれば setter としても働く
    - つまり`a = compVal`のように値を取り出すだけではなく、`compVal = 12`のように値を代入することもできるということ
