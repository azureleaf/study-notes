## Commands

1. 最重要で毎日つかうやつ
    - `git add -A`
    - `git commit -m "feat:"`
    - `git push origin master`
    - `git pull origin master`
1. 覚えて当然のやつ
    - `git clone http://blahblah.git`
    - `git init`
    - `git log -n 3`
    - `git revert`
    - `git stash`
